⚠️ 注意事项

  • 整个 JSON 是一个数组，每道题是一个对象
  • 多选题 type 必须写 "multi"，不能写 "multiple"
  • 判断题 options 固定为 ["对", "错"]
  • 大题 options 为空数组 []
  • 填空题用 ____ 表示填空位置
  • 图片用 base64 编码的字符串（可选）

🐍 Python 图片识别转 JSON

# 安装依赖：pip install pytesseract pillow

import pytesseract
from PIL import Image
import json
import re

def image_to_json(image_path):
    """识别图片中的题目并转为JSON"""
    # 1. 读取图片并识别文字
    img = Image.open(image_path)
    text = pytesseract.image_to_string(img, lang='chi_sim+eng')

    # 2. 解析题目（示例：按题号分割）
    questions = []
    # 这里需要根据实际图片格式调整解析逻辑
    # 示例：假设题目格式为 "1. 题目内容 A.xxx B.xxx C.xxx D.xxx"

    # 简单的正则匹配示例
    pattern = r'(\d+)\.\s*(.+?)\s*A\.\s*(.+?)\s*B\.\s*(.+?)\s*C\.\s*(.+?)\s*D\.\s*(.+)'
    matches = re.findall(pattern, text, re.DOTALL)

    for match in matches:
        num, title, optA, optB, optC, optD = match
        questions.append({
            "type": "single",
            "title": title.strip(),
            "options": [
                f"A.{optA.strip()}",
                f"B.{optB.strip()}",
                f"C.{optC.strip()}",
                f"D.{optD.strip()}"
            ],
            "answer": "C"  # 需要手动设置正确答案
        })

    # 3. 保存为JSON文件
    with open('questions.json', 'w', encoding='utf-8') as f:
        json.dump(questions, f, ensure_ascii=False, indent=2)

    return questions

# 使用

📋 JSON 格式说明

一、type 值（题型）

single 单选题 answer 填单个字母如 "A"
multi 多选题 answer 填字母数组如 ["A","C"]
judge 判断题 options 固定 ["对","错"] / answer 填 "A"(对) 或 "B"(错)
fill 填空题 answer 填字符串或字符串数组
essay 大题 支持文字+画图+图片 / options 为空数组 []

二、各题型完整示例

【single / 单选题】
{
  "type": "single",
  "title": "TCP/IP协议中，IP协议位于哪一层？",
  "options": ["应用层", "传输层", "网络层", "数据链路层"],
  "answer": "C"
}
【multi / 多选题】
{
  "type": "multi",
  "title": "以下哪些是合法的IP地址？",
  "options": ["192.168.1.1", "256.1.1.1", "10.0.0.1"],
  "answer": ["A", "C"]
}
【judge / 判断题】
{
  "type": "judge",
  "title": "子网掩码中，二进制的1代表网络部分。",
  "options": ["对", "错"],
  "answer": "A"
}
【fill / 填空题】
{
  "type": "fill",
  "title": "IPv4地址使用____位二进制数表示。",
  "answer": "32"
}
【essay / 大题】
{
  "type": "essay",
  "title": "请画出TCP三次握手的过程。",
  "answer": "参考答案内容"
}

三、字段说明

字段必填说明
type是题型标识：single/multi/judge/fill/essay
title是题目内容，填空题用____表示填空位置
options条件单选/多选/判断必填，大题填[]
answer是正确答案（单选填字母/多选填数组/判断填A或B/填空填字符串/大题填参考答案）
attachments否题目图片数组（base64），大题可选

四、注意事项

  • ⚠️ type 值必须是 single/multi/judge/fill/essay，不能写 multiple
  • ⚠️ judge 题 options 必须固定为 ["对","错"]，answer 只能填 "A" 或 "B"
  • ⚠️ fill 题 answer 可以是字符串或字符串数组（多个正确答案）
  • ⚠️ essay 题 options 为空数组 []
  • ⚠️ 整个 JSON 文件是一个数组，每道题是一个对象
